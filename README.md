# TEXT_TO_IMG

TEXT_TO_IMG는 입력한 텍스트를 이미지로 생성하고 다운로드할 수 있는 정적 웹앱입니다.

별도 서버, 빌드 도구, 프레임워크 없이 HTML, CSS, JavaScript만으로 동작합니다.

## 주요 기능

### 텍스트 이미지 생성

- 텍스트를 입력하면 canvas 미리보기에 즉시 반영됩니다.
- 입력한 텍스트는 이미지 중앙 영역에 렌더링됩니다.
- 긴 텍스트는 이미지 폭에 맞춰 자동 줄바꿈됩니다.
- 선택한 텍스트 일부에만 스타일을 적용할 수 있습니다.

지원하는 부분 텍스트 스타일:

- 볼드
- 밑줄
- 취소선

### 폰트 설정

지원 폰트:

- Noto Sans KR
- Pretendard
- 나눔 고딕
- Arial
- Georgia
- serif
- sans-serif

폰트 크기도 직접 입력할 수 있습니다.

### 텍스트 정렬

텍스트 정렬을 선택할 수 있습니다.

- 좌측 정렬
- 가운데 정렬
- 우측 정렬

### 색상 설정

브라우저 기본 컬러 피커 대신 프리셋 스와치와 HEX 입력을 사용합니다.

설정 가능한 색상:

- 배경색
- 글자색
- 테두리 색상

지원 HEX 형식:

- `#fed002`
- `fed002`
- `#fd0`
- `fd0`

### 배경색 없음

`배경색 없음`을 선택하면 투명 배경으로 이미지를 생성할 수 있습니다.

- PNG, WebP: 투명 배경 유지
- JPG: 투명 배경을 지원하지 않으므로 흰색 배경으로 처리

### 이미지 모양

이미지 모양을 선택할 수 있습니다.

- 네모
- 둥근 네모
- 원
- 캡슐
- 마름모

선택한 모양은 실제 다운로드 이미지에도 반영됩니다.

### 테두리

이미지 안쪽에 테두리를 포함할 수 있습니다.

설정 항목:

- 테두리 포함 여부
- 선 패턴
- 선 굵기
- 선 색상

지원 선 패턴:

- 실선
- 파선
- 점선

### 이미지 크기

표준 크기 3개와 직접 입력을 지원합니다.

표준 크기:

- 1200 x 630
- 1024 x 1024
- 800 x 450

직접 입력:

- 가로
- 세로

### 출력 배율

작은 이미지에서도 더 선명한 결과물을 만들 수 있도록 출력 배율을 지원합니다.

지원 배율:

- 1x
- 2x
- 3x
- 4x

예를 들어 이미지 크기가 `300 x 300`이고 출력 배율이 `2x`이면 실제 canvas와 다운로드 이미지는 `600 x 600`으로 렌더링됩니다.

### 이미지 다운로드

지원 확장자:

- PNG
- JPG
- WebP

파일명을 입력하지 않으면 기본 파일명 `TTI`로 다운로드됩니다.

예:

```text
TTI.webp
TTI.png
TTI.jpg
```

### UI 테마

UI 테마를 선택할 수 있습니다.

- 다크 모드: 검정 배경과 노란 글자 중심
- 라이트 모드: 흰 배경과 검정 글자/테두리 중심

### 마지막 입력값 저장

사용자가 마지막으로 설정한 입력값은 브라우저 `localStorage`에 저장됩니다.

다음에 같은 브라우저에서 사이트를 다시 열면 마지막 상태가 자동으로 복원됩니다.

저장되는 항목:

- 텍스트와 부분 스타일
- 파일명
- 폰트
- 폰트 크기
- 배경색
- 글자색
- 배경색 없음 여부
- 이미지 모양
- 텍스트 정렬
- 테두리 설정
- 이미지 크기
- 출력 배율
- 확장자
- UI 테마

## 기본값

앱 최초 실행 시 기본값은 다음과 같습니다.

- 입력 텍스트: `텍스트입력`
- 폰트: `나눔 고딕`
- 폰트 크기: `40`
- 배경색: `#000000`
- 글자색: `#fed002`
- 이미지 모양: `네모`
- 이미지 크기: `직접 입력`
- 가로: `300`
- 세로: `300`
- 출력 배율: `2x`
- 확장자: `WebP`
- UI 테마: `다크`

```text
TEXT_TO_IMG
├── index.html

├── README.md
```

### `index.html`

앱의 화면 구조를 담당합니다.

포함 요소:

- 설정 패널
- 텍스트 편집 영역
- 스타일 버튼
- 색상 스와치
- 이미지 모양 선택
- 테두리 설정
- 크기/확장자 설정
- 다운로드 버튼
- 미리보기 canvas
- 개발자 링크

### `style.css`

앱의 시각 스타일을 담당합니다.

포함 요소:

- 다크/라이트 테마 CSS 변수
- 미니멀 UI 레이아웃
- 설정 패널 스타일
- 스와치 스타일
- 이미지 모양 버튼 스타일
- 선 패턴 버튼 스타일
- 미리보기 영역 스타일
- 반응형 레이아웃

### `script.js`

앱의 동작과 canvas 렌더링을 담당합니다.

주요 역할:

- 입력값 수집
- 텍스트 스타일 적용
- 텍스트 줄바꿈
- canvas 렌더링
- 이미지 모양 clipping
- 테두리 렌더링
- 다운로드 처리
- HEX 색상 보정
- UI 테마 전환
- 마지막 설정값 저장 및 복원
